# 🌱 Crop Recommendation System

## 📌 Project Overview
This project is a web-based Crop Recommendation System that suggests the most suitable crop based on soil and environmental parameters such as nitrogen, phosphorus, potassium, temperature, humidity, pH, and rainfall.

---

## ⚙️ Tech Stack
- Python (Flask)
- MySQL Database
- HTML, CSS, JavaScript
- Machine Learning (if used)

---

## 🗄️ Database
- MySQL was used for storing soil and crop-related data.
- Database is exported as `database.sql` for easy import.

---

## 🚀 How to Run the Project

### 1. Install dependencies

pip install -r requirements.txt


### 2. Import database
- Open MySQL Workbench
- Import `database.sql`

### 3. Run the application

python app.py


### 4. Open in browser

http://127.0.0.1:5000


---

## 📊 Features
- Crop prediction based on soil data
- User-friendly web interface
- Database integration with MySQL
- Real-time input processing
